## 教程收藏取消收藏格式

网址：/course/fav

### 1.收藏

请求码：11008

请求格式：

|         |       |
| ------- | ----- |
| uid     | 用户ID  |
| authkey | 用户授权码 |
| cid     | 教程ID  |
| type    | 11008 |

返回值：11084:你已经收藏过此教程

11083：收藏教程成功

11082：服务器出错

11081：该教程无效

11080：授权码不正确



### 1.取消收藏

请求码：11009

请求格式：

|         |       |
| ------- | ----- |
| uid     | 用户ID  |
| authkey | 用户授权码 |
| cid     | 教程ID  |
| type    | 11009 |

返回值：11092:取消收藏成功

11093：你已经取消过此次收藏

11091：你没有收藏过此次教程

11082：服务器出错

11081：该教程无效

11080：授权码不正确
