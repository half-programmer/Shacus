| user_info信息 | 用户的相关信息         |
| ----------- | --------------- |
| ulikedN     | 粉丝数             |
| ulikeN      | 关注者的人数          |
| utel        | 电话              |
| uid         | 用户ID            |
| uname       | 用户姓名            |
| uimage      | 用户头像（暂未完成，默认为空） |
| usign       | 用户签名            |
| ubrithday   | 用户生日            |
