## 获取聊天所需用户信息

网址：/user/info

请求码：10705

请求格式：

|         |       |
| ------- | ----- |
| uid     | 用户ID  |
| authkey | 用户授权码 |
| type    | 10705 |
| seeid   | 查看人ID |

返回码：

10704：用户授权码不正确

10706：返回成功
