## 1.收藏约拍

| 参数      | 类型          | 描述/值             |
| ------- | ----------- | ---------------- |
| type    | String      | 收藏动作类型，此处值为10501 |
| uid     | Integer     | 用户id             |
| authkey | VARCHAR(32) | 用户认证码            |
| typeid  | Integer     | 约拍id             |

### 返回值示例：

## 10520：

```json
{
  "code": "10520", 
  "contents": "收藏成功"
}
```

## 10521:

```json
{
  "code": "10521", 
  "contents": "用户认证错误！操作失败"
}
```

## 10524：

```json
{
  "code": "10524", 
  "contents": "该约拍不存在！"
}
```
